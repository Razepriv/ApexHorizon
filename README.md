## Getting Started

### Prerequisites

- Node.js 18+
- PostgreSQL database
- Google AI API key

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Razepriv/ApexHorizon.git
cd ApexHorizon
```

2. Install dependencies:
```bash
npm install
```

3. Copy the environment file and fill in your values:
```bash
cp .env.example .env
```

4. Set up the database:
```bash
npm run db:push
```

5. Start the development server:
```bash
npm run dev
```
